# _разные слова и выражения

## 🔹 Выражения (Expresiones)

| Испанский | Русский перевод | Транскрипция |
|-----------|----------------|--------------|
| **gracias** | спасибо | ГР́А-сьяс |
| **muchas gracias** | большое спасибо | М́У-час ГР́А-сьяс |
| **perdona** | извини, прости | пер-Д́О-на |
| **sí, correcto** | да, верно | с́И, ко-Р́ЭК-то |
| **sí, verdad** | да, верно | с́И, бер-Д́АД |
| **no sé** | не знаю | но с́Э |
| **qué pena** | как жаль | ке П́Э-на |
| **qué interesante** | очень интересно | ке ин-те-ре-С́АН-те |
| **qué bien** | прекрасно | ке бьен |
| **qué bueno** | хорошо | ке Б́УЭ-но |
| **si tengo tiempo** | если есть время | с́И Т́ЭН-го ть-ЭМ-по |
| **sobre todo** | прежде всего / особенно / в основном | С́О-бре Т́О-до |
| **yo también** | я тоже | йо там-БЬЕН |
| **por ejemplo** | например | пор эх-ЭМ-пло |
| **comme il faut** | правильно, как надо | ком иль фут |
