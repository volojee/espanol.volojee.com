# Настоящее время, SER и ESTAR

## 🔹 Настоящее время (Presente de Indicativo) — краткий обзор

Для регулярных глаголов окончания:

| Тип глагола | -AR | -ER | -IR |
|-------------|-----|-----|-----|
| **yo** | -o | -o | -o |
| **tú** | -as | -es | -es |
| **él/ella/usted** | -a | -e | -e |
| **nosotros/as** | -amos | -emos | -imos |
| **vosotros/as** | -áis | -éis | -ís |
| **ellos/ellas/ustedes** | -an | -en | -en |

> Примеры: *hablar → hablo*, *comer → comes*, *vivir → vive*

---

## 🔹 Глагол **SER** (быть — постоянные характеристики)

### Сопряжение:
| Лицо | Форма |
|------|-------|
| yo | **soy** |
| tú | **eres** |
| él/ella/usted | **es** |
| nosotros/as | **somos** |
| vosotros/as | **sois** |
| ellos/ellas/ustedes | **son** |

### Когда использовать SER:
✅ **DOCTOR** — мнемоническое правило:

| Буква | Значение | Пример |
|-------|----------|--------|
| **D**escription | Описание | *Ella **es** alta.* (Она высокая) |
| **O**ccupation | Профессия | *Soy **médico**.* (Я врач) |
| **C**aracterística | Характер | *Él **es** amable.* (Он добрый) |
| **T**iempo | Время/дата | *Hoy **es** lunes.* (Сегодня понедельник) |
| **O**rigen | Происхождение | *Somos **de México**.* (Мы из Мексики) |
| **R**elación | Родство/принадлежность | *Ella **es** mi hermana.* (Она моя сестра) |

---

## 🔹 Глагол **ESTAR** (быть — временные состояния и местоположение)

### Сопряжение:
| Лицо | Форма |
|------|-------|
| yo | **estoy** |
| tú | **estás** |
| él/ella/usted | **está** |
| nosotros/as | **estamos** |
| vosotros/as | **estáis** |
| ellos/ellas/ustedes | **están** |

### Когда использовать ESTAR:
✅ **PLACE** — мнемоническое правило:

| Буква | Значение | Пример |
|-------|----------|--------|
| **P**osition | Позиция | *El libro **está** en la mesa.* (Книга на столе) |
| **L**ocation | Местоположение | *Madrid **está** en España.* (Мадрид в Испании) |
| **A**ction (Progressive) | Действие в процессе | *Estoy **estudiando**.* (Я учусь сейчас) |
| **C**ondition | Состояние | *Estoy **cansado**.* (Я устал) |
| **E**motion | Эмоция | *Ella **está** feliz.* (Она счастлива) |

---

## 🔹 Ser vs Estar: Сравнение на примерах

| Ситуация | Ser | Estar |
|----------|-----|-------|
| **Характер vs Настроение** | *Él **es** aburrido.* (Он скучный человек) | *Él **está** aburrido.* (Ему скучно сейчас) |
| **Профессия vs Временная роль** | *Soy profesor.* (Я учитель по профессии) | *Estoy de profesor.* (Я работаю учителем временно) |
| **Цвет (постоянный) vs Изменение** | *La nieve **es** blanca.* (Снег белый по природе) | *La camisa **está** blanca.* (Рубашка выстирана/чистая) |
| **Местоположение события** | *La fiesta **es** en mi casa.* (Вечеринка проходит у меня — событие) | *Mi casa **está** en el centro.* (Мой дом находится в центре — география) |

> 💡 **Простое правило**:
> **Ser** = кто ты есть (постоянно)
> **Estar** = где/как ты сейчас (временно)

---

## 🔹 Бонус: Частые ошибки и лайфхаки

❌ *~~Yo soy cansado~~* → ✅ **Estoy cansado** (усталость — временное состояние)
❌ *~~La manzana está verde~~* (если речь о цвете) → ✅ **La manzana es verde** (зелёный — постоянный признак)
✅ Но: *La manzana **está** verde* = яблоко неспелое (состояние!)

🎯 **Лайфхак**: Если можно добавить «сейчас» или «в данный момент» — скорее всего, нужен **estar**.
